## 地球管理系统

本项目以「地球」为背景，构建一个虚构的星球级综合治理与监控指挥平台，用于展示行星级数据看板、资源管理与生物圈管理等能力，侧重界面设计与交互体验，并不直接连接任何真实生产系统。

本项目的界面与交互，基于开源组件库 `neo-brutal-vue`（代码托管地址：`https://github.com/i-shl/neo-brutal-vue`）进行构建；整个项目代码由 AI 自动生成，主要用于演示该组件库在实际界面中的使用方式与整体展示效果。

---

### 一、项目定位

- **整体定位**：行星级综合管理中枢，可视化呈现地理、人口、资源、生态、运行状态等信息  
- **使用场景**：科幻氛围的演示大屏、原型展示、界面设计练习、数据可视化练习  
- **核心目标**：通过统一的视觉风格与多种图表组件，模拟一个「星球控制台」的工作状态

---

### 二、主要功能模块

- **一、监控总览大屏**
  - 全屏监控视图，顶部显示运行时间、当前系统时间等信息  
  - 中央区域为综合数据看板，展示人口、国家数量、海洋与陆地面积、社会稳定度、紧急预警数量等关键指标  
  - 多块图表矩阵，用于展示人口分布、资源储备、生态健康、经济情况等综合指标  
  - 右侧指令与预警区，用于展示突发事件、告警等级、待处理事项等

- **二、仪表盘视图**
  - 顶部统计卡片展示多项核心数据概览  
  - 多种图表组合展示不同维度的行星信息，例如人口分布、资源状态、环境状况、经济水平等  
  - 支持根据主题样式切换，整体视觉偏向科幻风格与块状新拟态设计

- **三、地理管理**
  - **洲级视图**：展示各大区域的面积、人口、环境、经济等综合信息  
  - **国家视图**：模拟国家级数据，包括人口、生产总值、资源、环境等指标  
  - **资源视图**：展示行星范围内的各类自然资源及其储量、分布与消耗情况

- **四、生物圈管理**
  - **物种视图**：展示不同物种的分布、数量、栖息环境与生存状态等信息  
  - **人类视图**：  
    - 顶部统计卡片：总人数、活跃人数、健康状况、平均年龄、平均收入、高学历比例等  
    - 多种图表：年龄结构、职业分布、健康指数变化、收入阶段分布等  
    - 列表数据：支持按姓名、国家、职业等条件搜索与筛选；可以根据状态、国家等条件进行过滤  
    - 操作入口：预留新增、查看详情、编辑、删除等交互入口（部分为演示用占位功能）

- **五、运行与空间活动**
  - 卫星视图：用于展示围绕行星运行的多个轨道目标及其状态、类型、所属轨道等信息  
  - 可以用作演示空间活动监控与调度的界面原型

---

### 三、界面与设计风格

- 整体视觉采用强烈的块状边框与阴影，形成新拟态与科幻控制台结合的视觉效果  
- 多处使用网格背景、扫描线、角标、徽章等元素，营造控制中心与监控大屏氛围  
- 列表、表格、卡片、图表等元素风格统一，文本多采用高对比度与较高字重，突出「控制台」感  
- 各管理页面之间在布局上保持一致性，便于快速理解与扩展

---

### 四、数据与交互说明

- 项目中的所有数据均为虚构示例，用于演示与练习，不代表任何真实世界数据  
- 多数视图支持通过图表与列表的方式同时观察同一类信息，方便从宏观与微观两个层面理解数据  
- 部分按钮与操作为演示用占位设计，目前仅用于展示交互意图，未接入真实后端服务  
- 通过主题与样式控制，可以对整体界面进行风格切换，适配不同展示环境（如大屏、桌面应用等）
